### [![GitHub license](https://img.shields.io/badge/caution-in%20progress-yellow.svg?style=flat-square)](http://opensource.org/licenses/MIT)

Toy Application For Pathfinders Club Managment

### Instalation

```sh
   $ npm install && bower install
```

### Run

Start Mongo DB:

```sh
  $ mongod
```

And, In another tab:

```sh
  $ npm start
```
